Nossa solução

A nossa solução é utilizar o sensor HC-SR04 para monitora o crescimento do milho, identificando o momento ideal para colheita usando dados para a geração de análises. O nosso sensor é capaz de medir distâncias utilizando ondas ultrassõnicas, identificando a altura das platas de forma automática e precisa. O sensor HC-SR04 será instalado em uma estrutura posicionado na lavoura, podendo analisar aproximadamente um hectare da plantação. Eles são distribuídos em pontos estratégicos, permitindo um melhor monitoramento do milho, fazendo a coleta dos dados e enviando informações para o sistema de análise.

Funcionamento do sensor

1- O sensor registra o momento exato em que o milho atinge uma determinada altura.

2- Quanto tempo demorou para atingir a altura determinada.

3- Serão enviados dashboard de gráficos e análise dos dados para uma melhor colheita.

Fluxo do nosso sistema de monitoramento

1- Medição da altura do milho:

Com o sensor bem posicionado, o monitoramento do desenvolvimento do milho já é inicializado.

2- Armazenamento dos dados:

Os dados coletados pelo nosso sensor serão armazenados para análise.

3- Dados enviados para o sistema:

Estes mesmo dados serão enviados a partir da uma API para o nosso sistema.

4- Envio de gráficos e análise para o cliente:

Com os dados tratados, serão entregues dashboards com gráficos e análise para o nosso cliente.

Benefícios do sistema

Aumento de produtividade:

O monitoramento contínuo da altura das plantas permite identificar o momento ideal para a colheita do milho, ajudando o agricultor a aproveitar o melhor potencial de sua plantação.

Redução de perdas:

Acompanhar o crescimento das plantas em tempo real ajuda a evitar colheitas precoces ou tardias, que podem trazer prejuízos financeiros.

Melhor tomada de decisões:

Os dados são coletados, armazenados e transformados em gráficos e relatórios, permitindo o acompanhamento da lavoura e a comparação de diferentes colheitas, ajudando em tomadas de decisões mais concretas.
